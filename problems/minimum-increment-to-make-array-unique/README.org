Given an array of integers A, a move consists of choosing any A[i], and incrementing it by 1.

Return the least number of moves to make every value in A unique.

Example 1:
#+BEGIN_EXAMPLE
Input: [1,2,2]
Output: 1
Explanation:  After 1 move, the array could be [1, 2, 3].
#+END_EXAMPLE

Example 2:
#+BEGIN_EXAMPLE
Input: [3,2,1,2,1,7]
Output: 6
Explanation:  After 6 moves, the array could be [3, 4, 1, 2, 5, 7].
It can be shown with 5 or less moves that it is impossible for the array to have all unique values.
#+END_EXAMPLE
 
Note:

1. 0 <= A.length <= 40000
- 0 <= A[i] < 40000

- Solution: Maintain a seed for each element

#+BEGIN_SRC go
// https://code.dennyzhang.com/minimum-increment-to-make-array-unique
// Basic Ideas: hashmap + greedy
//
// Once sorted, the final result of each item can be determined.
//
// Complexity: Time O(n*log(n)) Space O(n)
func minIncrementForUnique(A []int) int {
    if len(A) == 0 {
        return 0
    }
    sort.Ints(A)
    res, need := 0, A[0]
    for i, _ := range A {
        // need to reset the seed
        if A[i] >= need {
            need = A[i]+1
        } else {
            // reuse the seed
            res += need-A[i]
            need++
        }
    }
    return res
}
#+END_SRC

- Solution: Keep rolling over the debts

#+BEGIN_SRC go
// https://code.dennyzhang.com/minimum-increment-to-make-array-unique
// Basic Ideas: hashmap + greedy
//
// Increase items one by one
//
// Complexity: Time O(n) Space O(n)
func minIncrementForUnique(A []int) int {
    m := map[int]int{}
    for _, v := range A {
        m[v]++
    }
    res := 0
    max := 40000
    for v:=0; v<max; v++ {
        if m[v] <= 1 {
            continue
        }
        moves := m[v]-1
        res += moves
        m[v+1] += moves
    }
    // If it hits the biggest
    if m[max] > 1 {
        v := m[max]-1
        res += (v*(v+1))/2
    }
    return res
}
