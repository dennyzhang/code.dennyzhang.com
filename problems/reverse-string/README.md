Reverse String  

---

Write a function that takes a string as input and returns the string reversed.  

Example:  
Given s = "hello", return "olleh".

    ## Blog link: https://code.dennyzhang.com/reverse-string
    class Solution(object):
        def reverseString(self, s):
    	"""
    	:type s: str
    	:rtype: str
    	"""
    	return s[::-1]
